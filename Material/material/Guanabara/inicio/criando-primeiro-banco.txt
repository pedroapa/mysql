Os tipos de dados tem familiias e eles são sub-divididos

		Tipos Primitivos
Numérico(inteirio, real, logico)
Data/tempo
Literal (Caractere, texto, binario, coleçoes)
Espacial

Cada um desse tipo tem precições diferentes

O tipo inteiro tem varios tipos de precisoes diferentes como -> TinyInt, SmallInt, Int, MediumInt, BigInt
A diferenças entre eles é quantos Bytes vão utilizar para armanezar esse dado
TinyInt utiliza muito menos byte para armanezar do que o BigInt
ex: para cadastrar idade voce não precisa utilizar um "int", como uma pessoa vive normal até 100, um TinyInt já é o suficiente

#literal
O caractere possui a precisão char e varchar
A diferença que o char e varchar
se voce dizer um char com 30 letras, vai ficar reservado 30 letras, mas se voce utilizar apenas duas, vai desperdiçar o resto
O varchar seria reserver espaço para até 30 letras,  ele só utiliza o que precisa(se sobrar ele não vai ocupar esse espaço)

Qual a diferença entre o tipo CARACTER do TEXTO
Texto é para quando ser quer guardar  textos grandes, como descrição de uma pessoa
quando voce quer  pequeno texto como nome de alguem, endereço, telefone use o Caractere

No binario tem blod que da para armanezar uma imagem, mesmo que não seja recomendado guardar imagens dentro de banco de dados

No tipo Coleção temos precisão Enum e Set - onde neles voce pode configurar quais são os valores permitidos
